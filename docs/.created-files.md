## Key Features

### Correct Syntax Usage
- **Octaves**: Dots before/after notes (.S for lower, S. for upper)
- **Groups**: Square brackets [R G], not parentheses
- **Spaces**: Commas (,) for karvais, semicolon (;) for double space
- **Commands**: \beatDuration(4), \cycle("|4|2|2|"), \line, \role
- **Embellishments**: ~, ~/, ~\, ~w, ~^, etc.
